# Wayang Lingua Nusantara

Protected GitHub Pages app for Wayang-based EFL learning.

## Access
- Students and lecturers create accounts using Register.
- Administrator enters through Admin using PIN `JS2026`.

## Included
- Protected landing page
- Student, lecturer, and admin dashboards
- 16-week ready-use EFL materials
- Online Room Center with open join, breakout rooms, chat, notes, attendance, and screen-share preview
- AI tutors from Wayang characters
- Editable profiles with photo upload
- Assessment center and role-based reports
- JSON/CSV export

## GitHub Pages
Upload all root files to your repository and enable Settings > Pages > Deploy from branch > main > /root.
