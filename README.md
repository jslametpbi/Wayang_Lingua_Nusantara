# Wayang Lingua Nusantara

Protected, role-based Wayang-EFL learning app for GitHub Pages.

## Access
- Students and lecturers register first, then login.
- Administrator uses the Admin tab with PIN: JS2026.

## Included
- 16 complete meetings with objectives, listening, reading, speaking, writing, quiz, rubric, homework, and room agenda.
- AI Tutors: Semar, Gareng, Petruk, Bagong, Dalang AI, Arjuna, Srikandi, Bima, Gatotkaca, Kresna.
- Online Room Center with room creation/editing, open join links, breakout rooms, chat, attendance, notes, and screen-share preview.
- Student, lecturer, and admin dashboards.
- Editable profiles with photo upload.
- Chat, Forum, Help Desk.
- Reports with JSON/CSV export and print.
