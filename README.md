# Wayang Lingua Nusantara V9 Executive 6D

A protected, role-based Wayang-EFL learning app for GitHub Pages.

## Access
- Student/Lecturer: Register first, then login.
- Admin: use PIN `JS2026`.

## Included
- One-screen protected landing page
- Student, lecturer, and admin dashboards
- 16 complete weekly learning meetings
- Interactive AI Tutor Studio with write, respond, listen, stop, clear, save, and speech input
- Material Builder for lecturer/admin
- Online Room Center with create/edit, open join, breakout rooms, chat, notes, attendance, and screen-share preview
- Reports for all roles with JSON/CSV export and print
- Certificate Studio
- Editable profile and photo upload
- Chat, Forum, Help Desk

## Deploy
Upload all files in this folder directly to the root of your GitHub repository, then enable GitHub Pages from the main branch root.
