# Wayang Lingua Nusantara V12 PRO REAL

A protected, GitHub Pages-ready Wayang-based EFL learning app.

## Access
- Student/Lecturer: Register first, then login.
- Admin: choose Admin tab and enter PIN `JS2026`.

## Included
- One-screen protected landing page.
- Role-based Student, Lecturer, and Admin access.
- 16 complete Wayang-EFL meetings.
- Listening, speaking, reading, writing, vocabulary, quiz, rubric, and room agenda in each meeting.
- Interactive Wayang AI Tutor Canvas with 10 tutors.
- Respond, listen, stop, clear, and save buttons.
- Material Builder.
- Assessment Builder.
- Online Room Center with create/edit/join/chat/attendance/breakout tools.
- Chat, Forum, Help Desk, Logout.
- Editable profile and photo upload.
- Reports with JSON/CSV export and print.
- Certificate Studio with verification ID and QR-style mark.

## GitHub Pages Deployment
Upload all files inside this folder directly to the repository root. Do not upload the folder itself. Then go to Settings → Pages → Deploy from branch → main → /root → Save.
