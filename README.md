# Wayang Lingua Nusantara

Exclusive 6D-style Wayang EFL learning app for GitHub Pages.

## Access
- Student/Lecturer: Register first, then login.
- Admin: use PIN `JS2026`.

## Included
- Protected landing/login screen.
- Separate Student, Lecturer, and Admin dashboards.
- 16 complete weekly EFL meetings.
- AI Tutor Theatre with 10 Wayang mentors.
- Online Room Center with create/edit, open join, breakout labels, chat log, notes, attendance behavior, meeting links, and screen-share preview.
- Material Builder for lecturers/admin.
- Assessment, reports, certificate, profile photo upload.
- JSON/CSV exports.

## Deployment
Upload all root files to a GitHub repository and enable GitHub Pages from main/root.
