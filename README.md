# Wayang Lingua Nusantara

AI-powered Wayang storytelling app for EFL skills, intercultural communication, online rooms, AI-style mentor feedback, and research-ready learning reports.

## Access
- Student/Lecturer: register first, then login.
- Admin: use Admin PIN `JS2026`.

## Main Features
- Protected landing page; no materials are exposed before login.
- Separate dashboards for student, lecturer, and admin.
- Complete 16-week Wayang-based EFL materials.
- Listening, speaking, reading, writing, quiz, assessment, and portfolio tasks.
- 10 AI Wayang tutors: Semar, Gareng, Petruk, Bagong, Dalang AI, Arjuna, Srikandi, Bima, Gatotkaca, and Kresna.
- Online Room Center with available room templates and editable links.
- Editable student/lecturer profile with photo upload.
- Chat, Forum, and Help Desk buttons.
- CSV/JSON reports and printable certificate/report.

## GitHub Pages Deployment
Upload all files in this folder directly to the repository root, then enable GitHub Pages from `main` branch `/root`.

## Important Cache Note
If the old version still appears after deployment, open the site in an incognito window or clear site data for the GitHub Pages URL. This package disables old service-worker caching, but browsers may keep an older service worker until a hard refresh.
