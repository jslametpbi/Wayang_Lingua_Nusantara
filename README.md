# Wayang Lingua Nusantara Integrated Static Edition

This package is a GitHub Pages-ready integrated interface implementing the requested core systems as a local-first SPA: LMS, AI Tutor Platform, Assessment Management, Virtual Classroom, Analytics, Certificate Verification, Research Passport, and Admin Console.

Admin login:
- Email: admin@wayanglingua.id
- Password: Admin@JS2026

Student/Lecturer accounts can be registered from the protected landing page.

Notes: This is a static local-first implementation using localStorage. For multi-user production deployment, connect the included data model to PostgreSQL/Supabase and a backend API.
