WAYANG LINGUA NUSANTARA — GITHUB PAGES DEPLOYMENT GUIDE
Copyright © Dr. Joko Slamet

1. Unzip the package.
2. Open your GitHub account.
3. Create a new repository, for example: Wayang-Lingua-Nusantara.
4. Upload everything inside this folder directly to the repository root.
5. Go to Settings > Pages.
6. Select Deploy from a branch.
7. Select main branch and /root folder.
8. Click Save.
9. Wait until GitHub provides the live link.

No build command is needed. This is a pure static app.
No API key is needed. The app runs directly in the browser.
No demo data or default PIN is included.
