WAYANG LINGUA NUSANTARA — GITHUB PAGES DEPLOYMENT

1. Unzip this package.
2. Open the unzipped folder.
3. Upload the CONTENTS directly into the GitHub repository root:
   index.html, styles.css, app.js, manifest.json, sw.js, assets/, README.md.
4. In GitHub: Settings > Pages > Deploy from branch > main > /root > Save.
5. Wait until GitHub Pages finishes deployment.
6. Open the app in Incognito first to avoid old browser cache.
7. Admin access: click Admin tab and use PIN JS2026.

Important: do not upload the ZIP file itself and do not upload a parent folder containing the files.
